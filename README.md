## Tech Stack

- Framework: Next.js
- Styling: Tailwind CSS
- UI Library: Shadcn UI
- LLM: Gemini Generative AI, Vercel AI SDK
- Vector DB: Pinecone

## Getting Started

```
npm install
npm run dev
```

## Project Structure

- src: next.js source code

  - app
  - components
  - utils

- scripts: python scripts for data processing + adding data to vector DB
